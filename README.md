# 开源图书(O-LIB)

**O-LIB** 是一款面向PC端的免费开源软件。

## 功能简介

- **开源软件**：软件完全开源，持续进行优化，且无广告干扰。
- **便捷搜索**：提供快速方便的搜索功能，帮助用户迅速找到所需图书。
- **自定义存储**：用户可以自定义图书文件的存储位置，便于管理和访问。
